# How to use the accelerator

## For students

Students can ask homework-related questions in natural language. The agent will respond according to the active pedagogy policy and will prefer hints, guided steps, or worked examples instead of direct solutions when that is the configured behavior.

## For professors

Professors can use the portal to:

- adjust help style and step limits
- toggle direct-answer behavior and citation requirements
- review the active knowledge sources used by the toolbox
- publish an updated toolbox version when new knowledge is added

## Local workflow

1. Build the agent and tests.
2. Update the environment variables and the policy JSON.
3. Run the portal build.
4. Deploy with the Azure deployment scripts once the environment is ready.
